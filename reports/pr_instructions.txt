## Git Commands

### To create the PR (if using GitHub CLI):

```bash
# Ensure you're on final-deliverable branch
git checkout final-deliverable

# Push the branch
git push origin final-deliverable

# Create PR
gh pr create --base main --head final-deliverable --title "Final deliverable: fixes, tests, docs" --body-file reports/pr_instructions.txt
```

### To create the tag:

```bash
git tag -a v1.0-final -m "v1.0-final - final deliverable"
git push origin v1.0-final
```

### If GitHub CLI is not available:

1. Push the branch:
   ```bash
   git push origin final-deliverable
   ```

   - Select `final-deliverable` as source and `main` as target
   - Use the PR description from above
   - Create the PR
