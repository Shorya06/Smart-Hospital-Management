This directory contains all test reports, coverage data, and validation artifacts for the Smart Hospital Management System v1.0-final.

- `backend_coverage_final.txt` - Final backend coverage report (100%)
- `coverage_analysis.txt` - Coverage analysis
- `coverage_progress.md` - Coverage progress tracking

## Usage

These reports are generated automatically when running:
- Backend tests: `pytest --cov=hospital_app --cov-report=html`
- Frontend tests: `npm test -- --coverage`
- Demo script: `./demo.sh` or `demo.bat`
